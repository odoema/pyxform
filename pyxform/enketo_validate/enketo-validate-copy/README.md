Enketo Validate 
==============

_Validate XForms using Enketo's form engine_

### Install

1. install nodeJS 6+
2. clone repo (TODO: publish on npm)
3. `npm install --production`


### Use

```bash
> ./validate ~/myform.xml
```

### Help
```bash
> ./validate --help
```
